# Hafta 1: Python'a Giriş

## Konu Özeti

Bu hafta, Python programlama diline giriş yapılır. `print()` fonksiyonu, tırnak kullanımı ve temel veri tipleri öğrenilir.

---

## Öğrenme Hedefleri

- Python'un temel felsefesini anlamak
- `print()` fonksiyonunu kullanmak
- Tek tırnak, çift tırnak ve üç tırnak farkını kavramak
- Temel veri tiplerini tanımak: `string`, `integer`, `float`
- `type()` ve `len()` fonksiyonlarını kullanmak

---

## Konu Başlıkları

### 1.1 Print Fonksiyonu
```python
print("Merhaba Dünya!")
print("İstanbul'un 5 günlük hava durumu tahmini")
```

### 1.2 Tırnak Kullanımı
- Tek tırnak: `'metin'`
- Çift tırnak: `"metin"`
- Üç tırnak: `"""uzun metin"""`

### 1.3 Veri Tipleri
| Tip | Örnek | Açıklama |
|-----|-------|----------|
| `string` | `"Züber Doğan"` | Karakter dizisi |
| `integer` | `1978` | Tam sayı |
| `float` | `73.3` | Ondalıklı sayı |

### 1.4 Gömülü Fonksiyonlar
- `type()`: Veri tipini gösterir
- `len()`: Karakter sayısını verir
