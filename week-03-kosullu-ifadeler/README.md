# Hafta 3: Koşullu İfadeler

## Konu Özeti

Bu hafta, Python'da koşullu ifadeler (if-elif-else) ve karşılaştırma/mantıksal operatörler öğrenilir.

---

## Öğrenme Hedefleri

- if, elif, else yapılarını kullanmak
- Karşılaştırma operatörlerini uygulamak: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Mantıksal operatörleri kullanmak: `and`, `or`, `not`
- İç içe koşullu ifadeler yazmak

---

## Konu Başlıkları

### 3.1 if-elif-else Yapısı
```python
if koşul:
    # koşul doğruysa çalışır
elif diğer_koşul:
    # ilk koşul yanlış, bu doğruysa çalışır
else:
    # hiçbiri doğru değilse çalışır
```

### 3.2 Karşılaştırma Operatörleri
| Operatör | Anlamı |
|----------|--------|
| `==` | Eşit |
| `!=` | Eşit değil |
| `<` | Küçük |
| `>` | Büyük |
| `<=` | Küçük eşit |
| `>=` | Büyük eşit |

### 3.3 Mantıksal Operatörler
| Operatör | Anlamı |
|----------|--------|
| `and` | Ve (her iki koşul doğru) |
| `or` | Veya (en az biri doğru) |
| `not` | Değil (tersine çevirme) |
